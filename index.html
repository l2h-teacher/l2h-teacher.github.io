该不该送孩子去美国留学

发表于 2021-10-26 分类于 观点

在这件事上家长需要权衡利弊、慎重决策，多考虑孩子多特点。

诺贝尔物理学奖得主、中国科学院院士杨振宁的看法：

  1. 国内的中小学教育并不比美国差
  2. 把中学生送去接受美国教育，是一件非常危险的事
    美国崇尚自由，尊重学生个性化发展，实际上也是一个放纵的是会，有些学生染上了恶习，甚至有很多小学生吸毒，如果把未成年孩子送到美国，会面临学坏的危险.

  4. 美国太放任，有没有好处呢？有，非常聪明的，90分以上的小孩在美国占便宜。
  5. 为什么呢？因为他不需要规规矩矩念书，他自己就都可以吸收进去，所以他就海阔天空，他的发展，尤其最聪明的人在美国容易发展。

聊聊跳绳

发表于 2021-01-07 分类于 体育运动

跳绳简介

  • 跳绳是一种古老的娱乐活动
  • 跳绳是一项很有效的有氧运动
  • 跳绳是拳击运动员的常规训练科目
  • 新时代生存难度系数：中考竞赛科目

跳绳是一种古老的娱乐活动。唐朝称“透索”，宋称“跳索”，明称“跳百索”、“跳白索”、“跳马索”，清称“绳飞”，清末以后跳绳称作“跳绳”。跳绳是一项极佳的运动项目，能有效训练个人的反应和耐力。

跳绳是一项很有效的有氧运动，从运动量来说，持续跳绳10分钟，与慢跑30分钟相差无几，可谓是一项耗时少、耗能大的有氧运动。跳绳能增强人体心血管、呼吸和神经系统的功能。

跳绳可以结实全身肌肉，消除臀部和大腿上的多余脂肪并能使动作敏捷、稳定身体的重心。跳绳训练人的弹跳、速度、平衡、耐力和爆发力，同时可培养准确性、灵活性、协调性。

注意事项

  • 跳绳前热身10分钟。
  • 运动时选择场地或穿减震鞋。
  • 运动后注意拉伸降温和放松。
  • 禁忌：心脏疾病、高血压或严重疾病。
